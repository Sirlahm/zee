## ğŸ¨ Customization

### Colors
You can customize the color scheme by editing the CSS variables in `src/App.css`:

```css
:root {
  --blush-pink: #FFD6E8;
  --rose-red: #FF6B9D;
  --deep-rose: #E84393;
  --cream: #FFF5F0;
  --warm-gold: #FFD89B;
  --soft-gold: #F4C87A;
  --text-primary: #8B4367;
  --text-secondary: #B76E94;
}
```

### Name
To change the name, edit the `<h1 className="name">` element in `src/App.jsx`

### Message
Customize the Valentine's message in the `message-container` and `response-message` sections in `src/App.jsx`
